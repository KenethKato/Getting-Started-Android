STEP 2: GO TO THE THEMES AND THEN CHANGE THEM TO MATERIAL DESIGN THEMES

** Change the parent theme

parent="Theme.MaterialComponents.Light.NoActionBar"

STEP 3: ADD 3 EMPTY FRAGMENTS

** RC Layouts, add 3 fragments (blank fragment)
** Adding a fragment will add a Java class matching the fragment
